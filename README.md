# Анализ датасета Titanic

## Описание проекта

Этот проект представляет собой анализ датасета Titanic. Задача - предсказать, выжил ли пассажир Титаника на основе различных характеристик.

## Структура проекта

- `titanic_analysis.py` - основной скрипт с полным анализом
- `requirements.txt` - зависимости Python
- `README.md` - описание проекта
- Генерируемые изображения:
  - `data_visualization.png` - визуализация данных
  - `correlation_matrix.png` - матрица корреляций
  - `knn_parameter_tuning.png` - подбор параметров k-NN
  - `confusion_matrix_knn.png` - матрица рассогласования
  - `model_comparison.png` - сравнение моделей
  - `feature_importance.png` - важность признаков
